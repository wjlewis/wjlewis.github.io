Recent Work

  • tm is a Turing Machine editor and simulator. It's written in TypeScript using React, Redux, and Redux-Saga, and it features an easy to use drag-and-drop interface, unbounded tape, machine download and upload, and some clever tricks behind the scenes. Try it out at wjlew.is/tm.

  • ramo is a port of miniKanren to JavaScript. It allows one to program in a relational style.

  • blox is an isomorphic Redux experiment, and a “hello world” for networked multiplayer systems. Clients dispatch Redux actions to and receive updates from a common server over a WebSockets connection. See the repo for details or try it out here.

  • brief-cases brings discriminated unions to JavaScript in a tiny, easy-to-use package.
